# Análise e Pré-processamento de Dados

Projeto desenvolvido durante a disciplina de Ciência de Dados do curso de Engenharia de Computação.

O objetivo deste repositório é apresentar técnicas de análise exploratória, visualização e pré-processamento de dados utilizando Python e bibliotecas amplamente utilizadas na área de Data Science.

## Conteúdo

### Análise Exploratória de Dados

Realização de análises estatísticas e visuais para compreensão dos dados, identificação de padrões, distribuição das variáveis e relacionamento entre atributos.

Principais atividades:

- Estatísticas descritivas
- Análise de correlação
- Visualização de distribuições
- Identificação de padrões e tendências
- Geração de gráficos para apoio à interpretação dos dados

### Pré-processamento de Dados

Aplicação de técnicas utilizadas na preparação de dados para modelos analíticos e de Machine Learning.

Principais atividades:

- Tratamento de valores ausentes
- Detecção e tratamento de outliers
- Normalização e padronização de dados
- Codificação de variáveis categóricas
- Engenharia de atributos
- Redução de dimensionalidade utilizando PCA

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objetivo Acadêmico

Este projeto teve como finalidade aplicar conceitos de Ciência de Dados e preparação de dados, explorando técnicas utilizadas em fluxos reais de análise e modelagem.

## Estrutura do Repositório

- `analise_titanic_sofia.ipynb` — análise exploratória e visualização de dados.
- `pre_processamento.ipynb` — aplicação de técnicas de limpeza e preparação de dados.
